Edit Region: Custom Rectangle

This region is a rectangle, that spans specified ranges of latitude and longitude.

 

Min Latitude
Max Latitude

These fields contain the minimum and maximum latitude of the rectangle. Latitude can range from -90.0 to 90.0, with positive values denoting the northern hemisphere.

 

Min Longitude
Max Longitude

These fields contain the minimum and maximum longitude of the rectangle. Longitude can range from -180.0 to 180.0, with positive values denoting the eastern hemisphere.

A rectangle can span a maximum of 180 degrees of longitude. If you enter a range of longitudes that spans more than 180 degrees, the result is a rectangle that "goes around the other way" and crosses the date line. For example, if you enter minimum longitude -170.0 and maximum longitude 160.0, the result is a rectange that spans 30 degrees of longitude, from 160.0 to 180.0 and from -180.0 to -170.0.

 

Min Depth
Max Depth

These fields contain the minimum and maximum allowed depth of aftershocks and foreshocks, in kilometers. Any aftershocks or foreshocks whose depth lies outside this range are ignored.

The default values are -5.0 and 700.0, which is the largest depth range allowed by OpenSHA. It is strongly recommended that you not change these value.
