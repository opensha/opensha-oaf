Catalog Windows

When you load an earthquake catalog into the GUI (whether from Comcat, a catalog file, or an existing forecast), several windows appear on the right side of the GUI. The windows display the catalog in various ways.

You access the windows by clicking the tabs located in the top right area of the GUI. The windows are described below.

 

Catalog Tab

Displays the earthquake catalog as a text file. The file is shown in the catalog file format.

 

Epicenters Tab

Plots the earthquake epicenter locations on a map. Aftershocks are plotted in colors that indicate the time at which the aftershock occurred. Foreshocks are plotted in gray. Each earthquake is shown as an open circle, with larger earthquakes shown as larger circles. The aftershock search region is shown as a dashed circle.

If regional data is available, then the outer region is shown as a larger dashed circle. The display of regional data is useful to assess if the aftershock search region is enclosing the earthquake sequence, or if the aftershock search region should be adjusted.

 

Mag/Num Dist Tab

Shows a magnitude/number plot, with magnitude on the horizonal axis and number on the vertical axis. The plot is constructed by grouping aftershocks into magnitude bins 0.1 magnitude wide. Then the following are drawn:

  • Blue circles showing the number of aftershocks in each bin.
     
  • Black circles and lines showing the cumulative number of aftershocks as a function of magnitude.
     
  • A yellow dashed line, with slope equal to the selected Gutenberg-Richter b-value, positioned close to the black curve for comparison.
     
  • A green vertical dashed line, labeled "Mmaxc", at the magnitude of the bin that contains the largest number of aftershocks.
     
  • A cyan vertical dashed line, labeled "Mc", at the estimated magnitude of completeness.
     
  • A red vertical dashed line at the magnitude of the mainshock.

The b-value used to draw the yellow dashed line is the b-value from the Common Parameters panel; if you change b-value then the plot will be re-drawn accordingly.

The magnitude of completeness used to draw the cyan vertical dashed line is the value of Mc from the Common Parameters panel; if you change Mc then the plot will be re-drawn accordingly. By default, "Mc" is 0.5 magnitude larger than the "Mmaxc" magnitude.

 

Mag/Time Tab

Shows a magnitude/time plot, with time on the horizonal axis and magnitude on the vertical axis. Each earthquake is shown as a solid circle, colored to indicate distance from the epicenter. Earthquakes of larger magnitude are shown as larger circles.

The plot includes aftershocks, the mainshock, and foreshocks. When the plot is initially displayed, it begin 0.75 days before the mainshock, even though the GUI by default fetches foreshocks up to 60 days before the mainshock. If you want to see all the foreshocks, use the Set Axis control at the bottom of the window to adjust the minimum x-value.

 

Cumulative Num Plot Tab

Plots the cumulative number of aftershocks as a function of time, starting at the time of the mainshock, with time on the horizontal axis and number on the vertical axis.

Initially, the plot includes aftershocks whose magnitude is above the estimated time-independent magnitude of completeness. The magnitude of completness used in the plot is the value of Mc from the Common Parameters panel; if you change Mc then the plot will be re-drawn accordingly.

When forecasts are computed, the plot changes to include aftershocks whose magnitude is above the time-dependent magnitude of competeness Mc(t), and adds cumulative number plots obtained from the Reasenberg-Jones forecasts. The number of aftershcks is shown in black, while RJ sequence-specific is blue, RJ generic is green, and RJ Bayesian is red. The time-dependent magnitude of completeness Mc(t) is computed using the parameters from the Common Parameters panel. The RJ curves show the expected number of aftershocks whose magnitude is above Mc(t), as a function of time, as predicted by the RJ model. (ETAS number plots are shown on a separate tab.)
