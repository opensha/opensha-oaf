Edit RJ Parameters

This panel lets you set the parameter ranges for the Reasenberg-Jones search grid. The total number of points in the search grid is limited to a maximum of 200,000.

 

RJ a-value Range
RJ a-value Number

These fields control the range for the "a" parameter, which is the productivity. The productivity has a linear scale, so the grid points are equally spaced. You can enter the minimum and maximum values of the parameter, and the number of grid points for the parameter. If you want to fix the parameter at a single value, enter 1 for the number and set the minimum and maximum to the same value.

 

RJ p-value Range
RJ p-value Number

These fields control the range for the "p" parameter, which is the Omori exponent. The Omori exponent has a linear scale, so the grid points are equally spaced. You can enter the minimum and maximum values of the parameter, and the number of grid points for the parameter. If you want to fix the parameter at a single value, enter 1 for the number and set the minimum and maximum to the same value.

Note that the RJ code cannot compute a Bayesian forecast if the Omori exponent is allowed to vary.

 

RJ c-value Range
RJ c-value Number

These fields control the range for the "c" parameter, which is the Omori offset. The Omori offset has a linear scale, so the grid points are equally spaced. You can enter the minimum and maximum values of the parameter, and the number of grid points for the parameter. If you want to fix the parameter at a single value, enter 1 for the number and set the minimum and maximum to the same value.

Note that the RJ code cannot compute a Bayesian forecast if the Omori offset is allowed to vary. Also note that the Omori offset should probably have a log scale, but for historical reasons it doesn't.
