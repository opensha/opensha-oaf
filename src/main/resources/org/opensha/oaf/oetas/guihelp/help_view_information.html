Information Window

The information window is located on the right side of the GUI, and is accessed by clicking the Information tab. The information window is always available.

The information window shows the current status of the GUI. Its contents changes depending on what you are doing within the GUI. The contents may include:

  • Program version number.
     
  • Number of CPU threads available.
     
  • Memory usage.
     
  • Mainshock information (magnitude, time, location, name, etc.).
     
  • Aftershock search region.
     
  • Reasenberg-Jones maximum likelihood parameter values and standard deviations, for generic, sequence-specific, and Bayesian models.
     
  • ETAS maximum likelihood parameter values, for generic, sequence-specific, and Bayesian models.

The contents of the information window changes as you work with the GUI. New information replaces the previous contents of the information window, so the contents is always current.
