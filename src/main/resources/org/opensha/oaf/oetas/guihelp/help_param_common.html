These controls let you select a time-dependent magnitude of completeness, according to the Helmstetter model. You can also select a time-independent magnitude of completeness.

The F, G, and H fields contain the three Helmstetter parameters. The Mcat field contains the catalog magnitude of completeness during "normal" times.

You use the drop-down to select the magnitude of completeness model you want to use.

  • Enable - Enables the use of a Helmstetter time-dependent incompleteness model. You need to enter the desired Helmstetter parameters into F, G, and H, and you need to supply the catalog magnitude of completeness in Mcat. Note that, by convention, setting G equal to 100.0 produces a time-independent magnitude of completeness equal to the value of Mcat.
     
  • World Values - Enables the use of a Helmstetter time-dependent incompleteness model with standard values for the world catalog. The F, G, and H fields are deactivated and set equal to the world values. You need to supply the catalog magnitude of completeness in Mcat.
     
  • California Values - Enables the use of a Helmstetter time-dependent incompleteness model with standard values for the California catalog. The F, G, and H fields are deactivated and set equal to the California values. You need to supply the catalog magnitude of completeness in Mcat.
     
  • Constant - Mcat - Selects a constant time-independent magnitude of completeness equal to the value of Mcat. The F, G, and H fields are deactivated, and G is set equal to 100.0. You need to supply the catalog magnitude of completeness in Mcat.
     
  • Constant - Mc - Selects a constant time-independent magnitude of completeness equal to the value of Mc (which is described below). The F, G, and H fields are deactivated, and G is set equal to 100.0. You need to supply the catalog magnitude of completeness in Mc.

 

Mc

The estimated magnitude of completeness for the catalog that has been loaded into the GUI. When a catalog is loaded, the GUI initializes this field by applying a maximum curvature algorithm to the catalog. You can edit this value.

 

b

The Gutenberg-Richter b-value. When a catalog is loaded, the GUI initializes this field to a pre-programmed value (or to a value used in a prior forecast). The initial value is likely to be 1.0. You can edit this value.

 

Tools for Mc and b-value
Run tool for Mc and b-value

These controls provide access to tools for analyzing the loaded catalog and estimating the magnitude of completeness or b-value. You select the tool you want from the drop-down, and then press the button to launch the tool, which will open in a sub-panel. There are currently two tools available:

  • Max Likelihood b - Estimates b-value by applying a maximim likelihood algorithm.
     
  • b-Positive - Estimates b-value by applying the b-positive algorithm.

 

Fit Start Inset
Fit End Inset

By default, parameter fitting is done using all aftershocks from just after the mainshock until the end of the catalog. This time range is called the fitting interval.

Sometimes, you may want to shrink the fitting interval, so you don't use all those aftershocks. For example, if the sequence seems to change character at some point in time, you may want parameter fitting to start at that time instead of at the time of the mainshock. Or, if the latter part of catalog is inaccurate due to delays in processing seismic data, you may want parameter fitting to end some time before the end of the catalog.

The Fit Start Inset field specifies the start of the fitting interval, expressed as days after the mainshock. The Fit End Inset field specifies the end of the fitting interval, expressed as days before the end of the catalog. Thus, setting both fields to 0.0 yields the default fitting interval.

 

Forecast Magnitude Bins

This drop-down selects the set of magnitude bins that appear in the forecast table. The following options are available.

  • Auto - Lets the system select the set of magnitude bins. Currently, the default magnitude bins are 3.0, 4.0, 5.0, 6.0, and 7.0.
     
  • 3.0 to 5.0 - Selects magnitude bins 3.0, 4.0, and 5.0.
     
  • 3.0 to 6.0 - Selects magnitude bins 3.0, 4.0, 5.0, and 6.0.
     
  • 3.0 to 7.0 - Selects magnitude bins 3.0, 4.0, 5.0, 6.0, and 7.0.
     
  • 3.0 to 8.0 - Selects magnitude bins 3.0, 4.0, 5.0, 6.0, 7.0, and 8.0.
