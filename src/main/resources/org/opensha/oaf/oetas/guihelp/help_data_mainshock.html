Data Source Options: Mainshock Only

This panel lets you specify options for loading a mainshock from Comcat, without loading an earthquake catalog.

 

Use analyst options

If this box is checked, then the GUI examines the most recent forecast to see if it contains analyst-supplied options for forecast parameters, search region, injectable text, etc. If so, then those analyst-supplied options are used to initialize the GUI.

If this box is not checked, or if no analyst-supplied options are found, then the GUI is initialized to default values.
