Options Sent to the Servers

You can use the Analyst Parameters panel to send options to the AAFS servers. The servers remember the options you set, and use them in all future forecasts. Options apply only to the mainshock that you are currently working with.

When you send options to the servers, you can choose to send just basic options, or you can choose to send both basic options and custom parameters. You use the Use custom parameters checkbox in the Analyst Parameters panel to select whether or not you want to send custom parameters.

Before sending basic options, you need to load data into the GUI; it is sufficient to load just the mainshock. However, before sending custom parameters you must not only load data into the GUI, but also fit forecast parameters.

The Analyst Parameters panel has the ability to save options to a file instead of sending them directly to the AAFS servers. You can examine the file to see what options would be sent, and you can use the File Operations panel to send the options in the file to the servers.

 

Basic Options

All basic options are located in the Analyst Parameters panel. They are:

  • Automatic forecast option (normal, enable, or disable).
     
  • Forecast duration (how long the automatic system issues forecasts).
     
  • Injectable text.
     
  • Analyst remark.
     
  • Event-sequence option (auto, update, ignore, or delete) and lookback time.

 

Custom Parameters

If you send custom parameters then, in addition to the basic options, the following additional parameters are sent to the servers:

  • Aftershock search region (from the Edit Data Source panel, or from a loaded catalog file).
     
  • Magnitude of completeness parameters (from the Common Parameters panel).
     
  • RJ b-value parameter (from the Common Parameters panel).
     
  • Start and end inset times (from the Common Parameters panel).
     
  • Forecast magnitude bins (from the Common Parameters panel).
     
  • RJ parameter search grid (from the RJ Parameters panel).
     
  • ETAS parameter search grid (from the ETAS Parameters panel).
     
  • ETAS model (auto, generic, sequence-specific, Bayesian, or custom) (from the ETAS Options panel).
     
  • ETAS prior (mixed/hierarchical, Gaussian, or uniform) (from the ETAS Options panel).
     
  • ETAS b-value and alpha (from the ETAS Options panel).
