Overview of Control Panels

The main control panels are located on the left side of the GUI's main window. You use the control panels to operate the GUI. Some portions of the control panels may be enabled or disabled depending on the GUI's configuration or current state.

The following table lists the main control panels, and summarizes their functions.

 

Control Panel Function
Data Source Loads data into the GUI. Data can come from Comcat, from a catalog file, or from a forecast download file. The data consists of a mainshock and its associated foreshocks and aftershocks.
Aftershock Parameters Adjusts the parameters that are used to compute forecasts. After setting parameters, you can fit the model parameters and compute forecasts.
More Tools Offers tools for a variety of functions, including:
  • Sending parameters to the server.
  • Turning forecasts on or off.
  • Entering injectable text.
  • Setting forecast duration.
  • Checking server status.
  • Deleting a forecast from PDL.
  • Sending a saved forecast to PDL.
  • Searching Comcat to pick an earthquake to work on.

 

Floating Panel Design

Many of the buttons on the main control panels open secondary control panels that offer additional functions. These secondary control panels "float" in front of the main window. You can move them around if they obscure a portion of the main window that you want to see.

In most cases, you can continue to interact with the main window while a secondary control panel is open. In particular, you can open more secondary control panels, so it is possible to have several secondary control panels open at the same time.

The floating panel design lets you have open the control panels that you want to use, while other control panels remain hidden away.

 

Help Viewer

The GUI includes a help viewer that displays built-in help pages. There are three ways to open the help viewer:

  • Most control panels have a button in the upper right corner that is labeled with a question mark. Pressing that button opens the help viewer and displays a page of help text for that control panel.
     
  • There is a Help Topics button on the GUI's main window. Pressing that button opens the help viewer and displays a page of links to help on various topics. In particular, there is help for all of the result windows that appear on the right side of the GUI's main window. There is also help for the command-line options that the GUI supports.
     
  • There is a Help button in the forecast table. Pressing that button opens the help viewer and displays a page of help for the forecast table and related controls.

The help viewer "floats" in front of the main window. You can move it around if it obscures a portion of the main window that you want to see. You can keep the help viewer open while interacting with the main window and with the secondary control panels.

At the bottom of the help viewer there are buttons labeled with left and right arrows. You can use these buttons to navigate backward and forward among the help pages that you have viewed.

Some of the help pages have links that you can click to view related pages. After clicking a link, you can press the back button to return to the original page.
