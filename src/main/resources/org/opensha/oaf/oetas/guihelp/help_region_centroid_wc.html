Edit Region: Centroid WC Circle

This region is a circle whose radius is a multiple of the Wells and Coppersmith rupture length. The center of the circle is determined using the centroid algorithm.

The centroid algorithm works by first drawing a circle around the mainshock's epicenter, and finding all the aftershocks that lie within the circle. Then, the center of the search region is determined by computing the centroid of the epicenters of all those aftershocks plus the mainshock.

 

WC Multiplier

The region's radius is equal to the value of this field multiplied by the Wells and Coppersmith rupture length.

For example, if the value of this field is 1.0 then the radius is equal to the Wells and Coppersmith rupture length. If it is 2.0 then the radius is double the Wells and Coppersmith rupture length, and so on.

 

Min Radius
Max Radius

These fields contain the minimum and maximum allowed radius of the region, in kilometers.

 

Min Depth
Max Depth

These fields contain the minimum and maximum allowed depth of aftershocks and foreshocks, in kilometers. Any aftershocks or foreshocks whose depth lies outside this range are ignored.

The default values are -5.0 and 700.0, which is the largest depth range allowed by OpenSHA. It is strongly recommended that you not change these value.
