Overview of Result Windows

Result windows are located on the right side of the GUI's main window. They appear below a series of tabs; you select a window to view by clicking on its tab. Result windows appear and disappear as needed while you work with the GUI.

The following table lists the tabs that may appear, and summarizes the result windows.

 

Tab Contents
Console Everything written to the console since the GUI was started.
Information Current status of the GUI, including configuration, mainshock information, search region, and parameter fitting results.
Epicenters Plots the earthquake epicenter locations on a map.
Mag/Num Dist Plot of aftershock magnitude/number distribution.
Mag/Time Plot Plot of earthquake magnitudes versus time of occurrence.
Cumulcative Num Plot Plots the cumulative number of aftershocks as a function of time.
Catalog Lists the loaded earthquake catalog in the catalog file format.
Model PDFs Plots the 1D and 2D marginal distributions of the fitted model parameters, for both RJ and ETAS.
ETAS Number Plots Plots the integral of the ETAS intensity function, as a function of time or as a function of transformed time.
Forecast Table Shows the RJ and ETAS forcasts in tabular form, and contains controls to save a forecast or send it to PDL.
