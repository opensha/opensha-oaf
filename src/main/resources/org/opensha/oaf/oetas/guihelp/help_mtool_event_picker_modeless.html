Event Picker

You can use the event picker to search for earthquakes in Comcat. Once you find an earthquake, you can select it for use in the GUI.

 

Available Earthquakes

This drop-down shows the results of a search. Earthquakes are listed in reverse chronological order (most recent first). Each entry shows the date, magnitude, and description of an earthquake. The drop-down is filled in when you click the Populate button.

You select the earthquake you want to work with from this drop-down. When you click on an earthquake, it becomes selected and its event ID appears in the Select button.

 

Minimum Magnitude

Use this drop-down to select the type of search that you want to perform. You can search for:

  • Any earthquake that has an OAF product.
     
  • An earthquake that has an OAF product and is at least a selected minimum magnitude. For example, you can search for earthquake of magnitude 6.0 and larger with an OAF product.
     
  • A recent earthquake of at least a selected magnitude, regardless of whether or not it has an OAF product. For example, you can search for magnitude 5.0 and larger earthquakes within the past month.

 

Search Region

Use this drop-down to select a geographic region to search. If you select World then the search looks for earthquakes anywhere in the world. If you nake a different selection then the search is limited to earthquakes in the selected region.

Note that the GUI usually looks in a somewhat larger area than the selected region. For example, if you select California then the GUI searches in the area covered by the California seismic networks (SCSN and NCSS), which extends somewhat beyond the borders of California.

 

Populate Event List

Press the button to contact Comcat and perform the search. The results of the search are inserted into the Available Earthquakes drop-down list.

A dialog box is displayed if no earthquakes are found that satisfy the conditions of the search.

 

Select

When an earthquake in the Available Earthquakes drop-down is selected, this button becomes active, and the event ID is inserted into the button. Press the button to transfer the event ID into the USGS Event ID field in the main panel.

 

Cancel

Press this button to close the event picker without choosing an earthquake.
