Data Source Panel

Use the data source panel to load data into the GUI. Data can come from Comcat, from a catalog file, or from a forecast download file. The data consists of a mainshock and its associated foreshocks and aftershocks.

 

USGS Event ID

Enter a Comcat event ID if you are loading data from Comcat. Otherwise, you can leave it blank.

 

Data Source Type

This drop-down list selects what type of data you want to load.

  • Comcat - Loads a mainshock from Comcat, calculates the aftershock search region, and then fetches the foreshocks and aftershocks.
     
  • Catalog File - Reads in a file that contains information about the mainshock, foreshocks, and aftershocks. The file can also contain an aftershock search region and other information.
     
  • Published Forecast - Loads a mainshock from Comcat, and then retrieves the earthquake catalog and forecast parameters from one of the existing aftershock forecasts.
     
  • Mainshock Only - Loads only a mainshock from Comcat, without loading an earthquake catalog. This can be used for operations that don't require a catalog, such as turning forecasts on or off, setting injectable text, setting forecast duration, and selecting event-sequence options.
     
  • Download File - Reads a forecast_data.json file, and obtains the mainshock, earthquake catalog, and forecast parameters from the file.

 

Data Source

Opens a sub-panel that lets you select various options for loading data. The set of options varies depending on the data source type.

 

Catalog

This button loads the data. The data includes a mainshock, and for all data types except Mainshock Only it also includes an earthquake catalog. The earthquake catalog is displayed in the right-hand side of the GUI in several tabs.

  • Catalog Tab - Displays the earthquake catalog as a text file.
     
  • Epicenters Tab - Plots the earthquake epicenter locations on a map. Colors indicate the time at which an earhquake occurred. The aftershock search region is shown as a dashed circle. If regional data is available, then the outer region is shown as a larger dashed circle.
     
  • Mag/Num Dist Tab - Groups aftershocks into magnitude bins 0.1 magnitude wide, and shows a plot with magnitude on the horizontal axis and number on the vertical axis (a magnitude/number plot). Also shows cumulative number of aftershocks, magnitude of completeness, and the Gutenberg-Richter relation.
     
  • Mag/Time Tab - Plots earthquakes with time on the horizontal axis and magnitude on the vertical axis (a magnitude/time plot). Colors indicate distance from the epicenter.
     
  • Cumulative Num Plot Tab - Plots the cumulative number of aftershocks as a function of time, with time on the horizontal axis and number on the vertical axis. Initially, the plot includes aftershocks whose magnitude is above the estimated time-independent magnitude of completeness. When forecasts are computed, the plot changes to include aftershocks whose magnitude is above the time-dependent magnitude of competeness, and adds cumulative number plots obtained from the Reasenberg-Jones forecasts. (ETAS number plots are shown on a separate tab.)

Depending on the selected options, the aftershock forecast parameters can be initialized to their default values, or they can be initialized to values obtained from an existing forecast or download file.
