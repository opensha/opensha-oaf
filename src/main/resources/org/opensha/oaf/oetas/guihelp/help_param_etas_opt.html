Edit ETAS Options

This panel lets you select options for computing ETAS forecasts.

 

ETAS Model
ETAS Model Weight

The drop-down list selects which ETAS model to use for computing the forecast.

  • Auto - Lets the system choose which model to use (generic, sequence-specific, or Bayesian).
     
  • Bayesian - Use the Bayesian model.
     
  • Sequence Specific - Use the sequence specific model.
     
  • Generic - Use the generic model.
     
  • Custom Weight - Activates the ETAS Model Weight field so you can enter a model weight. A weight of 0.0 selects the sequence-specific model, 1.0 selects the Bayesian model, and 2.0 selects the generic model. Weights between 0.0 and 1.0 interpolate between the sequence-specific and Bayesian models, while weights between 1.0 and 2.0 interpolate between the Bayesian and generic models.

 

ETAS Prior

The drop-down list selects which Bayesian prior to use for computing a Bayesian forecast.

  • Mixed/Hierarchical - Use the mixed-form, hierarchical priors from Barall et al.
     
  • Gaussian - Use Gaussian priors from van der Elst.
     
  • Uniform - Use a uniform prior.

 

Use common b-value
ETAS b

If the box is checked, then the ETAS forecast uses the same Gutenberg-Richter b-value as the Reasenberg-Jones forecast. That b-value comes from the Common Parameters panel.

If the box is not checked, then the ETAS b field is activated, and you can enter the b-value that you want to use for ETAS forecasts.

 

Use alpha = b
ETAS alpha

If the box is checked, then the ETAS parameter "alpha" is set equal to the Gutenberg-Richter b-value. This is the default.

If the box is not checked, then the ETAS alpha field is activated, and you can enter the value that you want to use for the ETAS parameter "alpha".
