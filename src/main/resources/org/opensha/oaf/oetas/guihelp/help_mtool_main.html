More Tools Panel

This panel provides a variety of tools, most of which let you interact with the AAFS servers, PDL, and Comcat.

 

Analyst Parameers

Opens a sub-panel that lets you adjust OAF server parameters. You can use this sub-panel to perform the following tasks:

  • Turn forecasts on or off.
  • Select how long the server should produce forecasts.
  • Enter injectable text or an analyst remark.
  • Select whether the server generates event-sequence products.
  • Select whether the server should use custom aftershock parameters.
  • Send analyst options to the server.
  • Save analyst options to a file.

 

Save Aftershock Catalog

Lets you save the aftershock catalog to a file. When you click this button, a file chooser opens to let you enter the filename. The catalog is saved in the catalog file format.

This button becomes active when you load data into the GUI that includes a catalog.

 

AAFS Server Status

Contacts the AAFS servers and displays their current status. For each server, the status is either Alive or Dead. Additional status information is displaeyed in the Console tab.

The GUI comes pre-configured to communicate with a specific server or pair of servers. If the servers require an encrypted connection, then you must install digital certificate files in the same directory as the GUI JAR file, and the GUI may prompt you to enter a PIN.

The GUI can also be pre-configured not to communicate with any server. In this case, clicking the button will produce error messages, because the GUI will attempt to contact a non-existent server.

 

Delete Product

Opens a sub-panel that lets you delete an OAF product from Comcat, and optionally also modify or delete an associated event-sequence product. The delete is sent directly to PDL.

 

File Operations

Opens a sub-panel that lets you perform the following operations on files:

  • Send a saved forecast_data.json file directly to PDL (which may also send an event-sequence product).
  • Send a saved forecast.json file directly to PDL, and optionally also an event-sequence product.
  • Send a saved analyst options file to the AAFS servers.

 

Event Picker

Opens a sub-panel that lets you search for earthquakes and pick an earthquake to work with. Searches can be constrained by magnitude and by geographic region.
